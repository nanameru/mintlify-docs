## AI Agent Presentation Framework へようこそ

AI Agent Presentation Frameworkは、複数のAIエージェントが協力してプレゼンテーション作成、リサーチ、コンテンツ生成を行うWebアプリケーションです。

### 主な機能

🎯 **自動スライド生成**
テーマやトピックを入力するだけで、AIが自動的にプロフェッショナルなプレゼンテーションスライドを作成します。

🔍 **ディープリサーチ**
複数の検索エンジンとAIを組み合わせて、包括的なリサーチを実行し、信頼性の高い情報を収集します。

🖼️ **画像生成**
テキストから高品質な画像を生成し、プレゼンテーションやドキュメントに直接組み込むことができます。

🌐 **ブラウザ自動化**
Webサイトの操作を自動化し、情報収集やスクリーンショット取得を効率的に行います。

💬 **マルチエージェントチャット**
複数のAIエージェントが協力して、複雑なタスクを段階的に解決します。

### 今すぐ始める

1. アプリケーションにアクセス
2. チャット画面でタスクを入力
3. AIエージェントが自動的に作業を開始
4. 結果をプレビューし、必要に応じて調整
5. 完成したコンテンツをエクスポート
