## はじめに

AI Agent Presentation Frameworkを使って、最初のAIエージェントを素早く立ち上げる方法を説明します。

### 前提条件

始める前に、以下の環境が準備されていることを確認してください：

- Node.js 18以上
- npm または yarn
- Git
- テキストエディタ（VS Code推奨）

## セットアップ

### 1. プロジェクトのクローン

<Steps>
  <Step title="リポジトリのクローン">
    まず、プロジェクトをローカル環境にクローンします：
    
    ```bash
    git clone https://github.com/your-org/ai-agent-presentation.git
    cd ai-agent-presentation
    ```
  </Step>
  
  <Step title="依存関係のインストール">
    プロジェクトディレクトリで以下のコマンドを実行：
    
    ```bash
    npm install
    # または
    yarn install
    ```
  </Step>
  
  <Step title="環境変数の設定">
    `.env.example` ファイルをコピーして `.env.local` を作成：
    
    ```bash
    cp .env.example .env.local
    ```
    
    必要なAPIキーを設定します：
    ```env
    OPENAI_API_KEY=your_openai_api_key
    DATABASE_URL=your_database_url
    ```
  </Step>
</Steps>

### 2. 開発サーバーの起動

環境設定が完了したら、開発サーバーを起動します：

```bash
npm run dev
# または
yarn dev
```

ブラウザで `http://localhost:3000` を開くと、アプリケーションが表示されます。

## 最初のエージェントを作成

### シンプルなチャットエージェント

以下のコードで基本的なチャットエージェントを作成できます：

```typescript
import { Agent } from '@/mastra/agent';

const chatAgent = new Agent({
  name: 'SimpleChat',
  description: 'シンプルなチャットエージェント',
  tools: [],
  prompt: `あなたは親切なアシスタントです。
           ユーザーの質問に分かりやすく答えてください。`
});

// エージェントの実行
const response = await chatAgent.execute({
  messages: [{ role: 'user', content: 'こんにちは！' }]
});

console.log(response);
```

### エージェントのテスト

作成したエージェントをテストするには：

<Tabs>
  <Tab title="API経由">
    ```bash
    curl -X POST http://localhost:3000/api/chat \
      -H "Content-Type: application/json" \
      -d '{
        "messages": [{"role": "user", "content": "こんにちは！"}]
      }'
    ```
  </Tab>
  
  <Tab title="UIから">
    1. ブラウザで `http://localhost:3000` を開く
    2. チャット画面でメッセージを入力
    3. Enterキーを押すか送信ボタンをクリック
  </Tab>
</Tabs>
