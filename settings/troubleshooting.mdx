## よくある問題と解決方法

### 一般的な問題

**チャット画面が応答しない**

**症状**: メッセージを送信しても反応がない

**原因と解決方法**:
1. **ネットワーク接続の確認**
   - インターネット接続を確認
   - ページを再読み込み（F5キー）

2. **ブラウザの問題**
   - キャッシュをクリア
   - 別のブラウザで試行
   - JavaScript有効化を確認

3. **サーバー負荷**
   - 少し時間をおいて再試行
   - シンプルな指示から開始

**スライド生成が途中で止まる**

**症状**: 「生成中...」から進まない

**解決方法**:
1. **指示の簡略化**
   ❌ 悪い例: "会社の全事業について詳細な50枚のプレゼンテーションを..."
   ✅ 良い例: "会社概要の5枚スライドを作成して"

2. **段階的な作成**
   - 第1段階: "まず構成案を考えて"
   - 第2段階: "最初の3枚を作成して"
   - 第3段階: "残りのスライドを追加して"

**画像が生成されない**

**症状**: 画像生成の指示に対して画像が表示されない

**確認事項**:
1. **指示の明確性**
   - 具体的な描写を含める
   - 不適切なコンテンツを避ける

2. **ブラウザ設定**
   - 画像表示がブロックされていないか確認
   - アドブロッカーの無効化

**ディープリサーチが完了しない**

**症状**: 長時間「検索中...」が続く

**対処法**:
1. **クエリの調整**
   ❌ 広すぎる: "世界について調べて"
   ✅ 適切: "2024年の日本のEV市場について"

2. **時間制限の理解**
   - 複雑なリサーチは10-15分かかる場合がある
   - 必要に応じて範囲を狭める

### ブラウザ固有の問題

**Chrome**
- **メモリ不足**: 他のタブを閉じる
- **拡張機能**: 一時的に無効化

**Safari**
- **WebSocket接続**: プライベートブラウジングを無効化
- **ストレージ**: サイトデータをクリア

**Firefox**
- **セキュリティ設定**: Enhanced Tracking Protectionを調整
- **開発者ツール**: コンソールでエラー確認

### パフォーマンス最適化

**応答速度の改善**

1. シンプルな指示から開始
2. 複雑なタスクを分割
3. 不要なタブを閉じる
4. 安定したネットワーク環境を使用

**メモリ使用量の削減**

1. 長時間使用後はページを再読み込み
2. 大きなファイルのエクスポート後はページをリフレッシュ
3. 他のアプリケーションを最小限に抑制

### エラーメッセージの対処

**「認証に失敗しました」**
- ページを再読み込み
- プライベートモードでの利用を確認

**「サーバーに接続できません」**
- ネットワーク接続を確認
- VPNを使用している場合は一時的に無効化

**「処理に失敗しました」**
- 指示内容を確認し、より具体的に再試行
- 不適切なコンテンツが含まれていないか確認

### 機能別トラブルシューティング

**スライド生成の問題**

**問題**: 期待したデザインにならない
**解決**: より具体的なデザイン指示を追加
"プロフェッショナルなブルー系、ミニマルデザインで"

**問題**: 文字が読みにくい
**解決**: フォントサイズや色の調整を指示
"タイトルを大きく、背景とのコントラストを強くして"

**問題**: スライド数が指定と異なる
**解決**: 明確な枚数指定
"必ず10枚で作成して、各スライドのテーマも指定"

**ディープリサーチの問題**

**問題**: 情報が古い
**解決**: 調査期間を明確に指定
"2024年の最新情報に限定して調査してください"

**問題**: 関連性の低い情報が含まれる
**解決**: より具体的な調査条件を設定
"BtoB企業に限定、従業員数100名以上の事例のみ"

**問題**: 情報量が多すぎる
**解決**: 要約や絞り込みを依頼
"重要な3つのポイントに絞って要約してください"

**画像生成の問題**

**問題**: イメージと異なる画像が生成される
**解決**: より詳細な描写を追加
"会議室、スーツ着用、ホワイトボード使用、明るい照明"

**問題**: 画質が低い
**解決**: 品質設定を指定
"高解像度、印刷用品質で生成してください"

**問題**: 商用利用に適さない要素が含まれる
**解決**: 制約条件を明確化
"著作権フリー、商用利用可能な要素のみ使用"

### 予防策

**効果的な指示の出し方**

1. **目的を明確にする**
   "○○向けの○○用資料として"

2. **制約条件を先に伝える**
   "時間制限○分、スライド○枚、予算○円"

3. **段階的にアプローチする**
   "まず概要を作成し、その後詳細を追加"

4. **参考例を提示する**
   "添付画像のようなスタイルで"

**作業環境の最適化**

1. **安定したネットワーク環境**
   - 有線接続の利用
   - 他のダウンロードを停止

2. **ブラウザの最適化**
   - 最新版への更新
   - 不要な拡張機能の無効化

3. **定期的なメンテナンス**
   - キャッシュの定期クリア
   - 履歴の整理

### お問い合わせ前のチェックリスト

問題が解決しない場合、以下を確認してからお問い合わせください：

- [ ] ブラウザとOS情報
- [ ] 実行した指示の内容
- [ ] エラーメッセージの正確な文言
- [ ] 問題発生までの手順
- [ ] 他のブラウザでの動作確認結果
- [ ] ネットワーク環境の確認
- [ ] 同様の操作での成功例の有無

### 緊急時の対処法

**重要な作業中にシステムが停止した場合**

1. **すぐに実行**: ページを再読み込みしない
2. **履歴確認**: ブラウザの履歴から直前の状態を確認
3. **段階的復旧**: 最後の成功時点から作業を再開
4. **バックアップ**: 重要な成果物は定期的にダウンロード

**データ損失を防ぐ方法**

1. **定期保存**: 長時間作業時は途中結果を保存
2. **複数形式**: PPTX、PDF、HTMLで複数保存
3. **クラウド同期**: 可能な場合はクラウドサービスと連携
