## 画像生成機能

### 基本的な画像生成

**生成方法**

チャット画面で画像生成の指示を入力します：

"[テーマ]の画像を生成してください。
スタイル：[リアル/イラスト/抽象的など]
用途：[プレゼンテーション/SNS/印刷物など]
サイズ：[16:9/1:1/4:3など]"

**生成例**

"ビジネスミーティングの画像を生成してください。
スタイル：プロフェッショナル、リアル
用途：プレゼンテーション用
サイズ：16:9（横長）
要素：会議室、ノートPC、チームディスカッション"

### 生成スタイル

**リアル系**
- 写真のようなリアルな画像
- ビジネスシーンに最適
- 人物、風景、商品撮影風

"商品発表会の様子をリアルな写真風で生成してください"

**イラスト系**
- 手描き風のイラスト
- 親しみやすい印象
- 説明資料や教材に最適

"新入社員研修の概念をイラストで表現してください"

**アブストラクト**
- 抽象的なデザイン
- コンセプチュアルな表現
- アート的な用途

"イノベーションの概念を抽象的なアートで表現してください"

**インフォグラフィック**
- データ可視化用
- グラフや図表スタイル
- 統計や比較に最適

"売上成長をインフォグラフィックで表現してください"

### 用途別最適化

**プレゼンテーション用**

"プレゼンテーション用のヘッダー画像を生成してください。
テーマ：デジタル変革
サイズ：横長（16:9）
色調：ブルー系、プロフェッショナル
テキストスペース：左側に余白を確保"

**ソーシャルメディア用**

"SNS投稿用の画像を生成してください。
テーマ：新商品告知
サイズ：正方形（1:1）
スタイル：キャッチー、明るい色調
テキスト：中央にタイトル配置予定"

**印刷物用**

"パンフレット表紙用の画像を生成してください。
テーマ：環境への取り組み
サイズ：A4縦（3:4）
解像度：高解像度
色調：グリーン系、自然的"

**Web用**

"Webサイトのバナー画像を生成してください。
テーマ：オンラインセミナー告知
サイズ：横長バナー（3:1）
ファイルサイズ：Web最適化
読み込み速度重視"

### 高度なカスタマイズ

**色調の指定**

- 明るい/暗い
- 暖色系/寒色系
- モノクロ/カラフル
- パステル/ビビッド

"企業ブランドカラー（青：#003366）を基調とした画像を生成してください"

**構図の指定**

- 中央配置/左右配置
- 上下分割/左右分割
- 背景ぼかし/全体フォーカス

"人物を右側に配置し、左側にテキストスペースを確保した構図で"

**要素の追加・除外**

"次の要素を含めてください：
- ノートパソコン
- グラフ
- チームメンバー3-4人
除外してください：
- 特定のブランドロゴ
- 個人を特定できる要素"

### 修正と調整

**部分的な修正**

"背景色をもう少し明るくしてください"
"人物の表情をより笑顔にしてください"
"テキストエリアを広くしてください"

**スタイルの変更**

"同じ構図でイラストスタイルに変更してください"
"よりビジネス的な雰囲気に調整してください"
"カジュアルな印象に変更してください"

**サイズ調整**

"正方形から横長（16:9）に変更してください"
"高解像度版を作成してください"
"ファイルサイズを軽量化してください"

### 実践的な活用例

**営業資料**

"顧客との商談シーンの画像を生成してください。
- 会議室での握手
- 契約書とペン
- プロフェッショナルな雰囲気
- 信頼関係を表現"

**研修教材**

"セキュリティ研修用のイラストを生成してください。
- フィッシングメールの危険性
- 分かりやすいアイコン
- 注意喚起の色調（赤・黄色）
- 教育的なスタイル"

**マーケティング**

"新サービスのコンセプト画像を生成してください。
- 革新性を表現
- 未来的なデザイン
- 明るく希望的な色調
- ターゲット層（20-40代）にアピール"

### ダウンロードと活用

**ファイル形式**

- PNG: 透明背景、高品質
- JPEG: 写真系、ファイルサイズ小
- SVG: ベクター形式、拡大縮小自由

**解像度選択**

- Web用: 72dpi、軽量
- 印刷用: 300dpi、高品質
- プレゼン用: 150dpi、バランス

**使用権利**

生成された画像は商用利用可能ですが、以下の点にご注意ください：

- 他者の著作権を侵害しない内容
- 公序良俗に反しない内容
- 特定の人物や企業を無断で使用しない
