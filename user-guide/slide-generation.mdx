## スライド生成機能の使い方

### 基本的なスライド作成

**1. テーマの設定**

スライド作成を始める前に、明確なテーマを設定します。

効果的なテーマ設定例：
- "デジタルマーケティングの基礎"
- "2024年度事業計画発表"
- "新入社員向けAI技術研修"

**2. 対象者の明確化**

誰に向けたプレゼンテーションかを指定することで、適切なトーンと内容レベルが選択されます。

対象者例：
- 経営陣向け → 戦略的視点、数値重視
- 顧客向け → 分かりやすさ、メリット重視
- 技術者向け → 詳細仕様、実装方法重視

**3. 時間と構成の指定**

"10分間のプレゼンテーション用に以下の構成で作成して：
1. 導入（2分）
2. 現状分析（3分）
3. 提案内容（4分）
4. まとめ（1分）"

### 高度なカスタマイズ

**ブランディング要素**

"企業カラーの青を基調とし、
ロゴを右上に配置したスライドを作成して"

**データ統合**

"添付のExcelデータを使用して、
売上推移グラフを含むスライドを作成して"

**多言語対応**

"英語版と日本語版の両方で
同じ内容のスライドを作成して"

### プレビューと調整

**リアルタイムプレビュー**

スライドが生成されると、右側のプレビューパネルで即座に確認できます。

プレビュー機能：
- フルスクリーン表示: 実際のプレゼン画面で確認
- スライド一覧: 全体の流れを把握
- 個別編集: 特定スライドの詳細調整

**修正指示**

"3枚目のスライドに円グラフを追加して"
"タイトルフォントをもう少し大きくして"
"背景色を白からライトグレーに変更して"

### エクスポートとダウンロード

**ファイル形式の選択**

- PPTX: PowerPointでの編集が可能
- PDF: 印刷や配布に最適
- HTML: Web上での表示や共有

**品質設定**

- 標準品質: 通常のプレゼンテーション用
- 高品質: 印刷や大画面表示用
- 軽量版: ファイルサイズを重視

### 実践的な活用例

**営業プレゼンテーション**

"新規顧客向けのサービス紹介プレゼンテーションを15分間で作成してください。
- 会社概要（2分）
- サービス特徴（5分）
- 成功事例（5分）
- 導入提案（3分）
企業カラーは青、信頼感を重視したデザインで。"

**研修資料**

"新入社員向けのセキュリティ研修資料を30分間で作成してください。
- セキュリティの重要性
- 基本的な脅威と対策
- 社内ルール
- 実践演習
分かりやすいイラストと具体例を多用してください。"

**業績報告**

"四半期業績報告用のプレゼンテーションを20分間で作成してください。
添付のデータを使用して：
- 売上実績
- 目標達成率
- 課題と改善策
- 次期計画
経営陣向けなので数値とグラフを中心に。"

### よくある質問

**Q: スライドの枚数は指定できますか？**
A: はい。"10枚のスライドで作成して"のように枚数を指定できます。

**Q: テンプレートは選択できますか？**
A: "プロフェッショナル/カジュアル/アカデミック"などのスタイルを指定できます。

**Q: アニメーションは追加されますか？**
A: 基本的なスライド切り替え効果が自動で追加されます。詳細な指定も可能です。
