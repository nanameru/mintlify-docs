## リサーチワークフロー

### 効率的な調査プロセス

このチュートリアルでは、ディープリサーチ機能を活用した効率的な調査プロセスと、その結果を活用した資料作成までの完全なワークフローを説明します。

### フェーズ1: 調査計画の立案

**調査目的の明確化**

"以下のリサーチ計画を立ててください：
テーマ：2024年のサステナビリティ経営
目的：CSR報告書作成のための情報収集
期限：1週間
成果物：包括的なレポートとプレゼンテーション"

**調査範囲の設定**

"調査範囲を以下に設定してください：
1. 国内外のトレンド（重点：日本企業の取り組み）
2. 法規制の動向（2024年施行予定の新制度）
3. 先進企業の事例（業界別トップ5社）
4. 投資家・ステークホルダーの期待
5. 今後の予測と提言"

### フェーズ2: ディープリサーチの実行

**基礎情報の収集**

"[Deep Research] サステナビリティ経営について以下を調査してください：
1. 定義と基本概念
2. 国際的なフレームワーク（GRI、SASB、TCFD等）
3. 日本政府の政策方針
4. 業界団体の取り組み指針"

**トレンド分析**

"[Deep Research] 2024年のサステナビリティ経営トレンドについて：
1. ESG投資の最新動向
2. 脱炭素・カーボンニュートラルの進展
3. サプライチェーン透明性の要求
4. 生物多様性保全の重要性拡大
5. サーキュラーエコノミーの実装"

**企業事例の調査**

"[Deep Research] サステナビリティ経営の先進企業事例：
1. 国内大手企業（トヨタ、ユニクロ、花王等）の取り組み
2. 海外企業のベストプラクティス
3. 中小企業の革新的な事例
4. スタートアップの新しいアプローチ"

### フェーズ3: 情報の整理と分析

**データの構造化**

"収集した情報を以下の形式で整理してください：
1. エグゼクティブサマリー（1ページ）
2. トレンド分析（図表付き）
3. 企業事例集（カテゴリ別）
4. 法規制マップ（時系列整理）
5. ベストプラクティス集"

**ギャップ分析**

"現在の調査結果と目標の間のギャップを分析してください：
不足している情報：
- 定量的なデータ
- 業界別の詳細分析
- 失敗事例と教訓
- コスト・ROI分析"

**追加調査の実行**

"[Deep Research] 不足している情報について追加調査：
1. サステナビリティ投資の市場規模とROI実績
2. 業界別（製造業、金融、小売）の取り組み比較
3. サステナビリティ経営の失敗事例と原因分析
4. 実装コストと効果測定の手法"

### フェーズ4: ビジュアル化と資料作成

**インフォグラフィックの作成**

"調査結果を視覚的に表現するインフォグラフィックを作成してください：
1. ESG投資の成長トレンド（2019-2024年）
2. 日本企業のサステナビリティ成熟度マップ
3. 業界別取り組み状況比較チャート
4. 2030年に向けたロードマップ"

**レポートの作成**

"包括的なリサーチレポートを以下の構成で作成してください：
1. エグゼクティブサマリー
2. 調査背景と目的
3. 市場動向とトレンド分析
4. 法規制環境の変化
5. 企業事例とベストプラクティス
6. 課題と機会の分析
7. 提言と次期アクション
8. 参考文献と出典"

### フェーズ5: プレゼンテーション作成

**ステークホルダー別資料**

"以下の対象者別にプレゼンテーションを作成してください：

**経営陣向け（15分）**：
- 戦略的意義と競争優位性
- 投資必要性とROI予測
- リスクと機会の分析
- 意思決定に必要な要素

**実務担当者向け（30分）**：
- 具体的な実施手法
- 他社の成功事例詳細
- 実装ステップとタイムライン
- 必要なリソースと体制

**投資家向け（20分）**：
- 市場機会とビジネス価値
- 競合との差別化
- 財務インパクト予測
- 長期価値創造ストーリー"

### フェーズ6: 品質チェックと最終化

**ファクトチェック**

"レポート内の全データについてファクトチェックを実行してください：
1. 数値データの出典確認
2. 引用情報の正確性検証
3. 最新性の確認（2024年データ優先）
4. 信頼性評価（一次ソース重視）"

**論理構成の確認**

"レポート全体の論理構成をチェックしてください：
1. 各章の関連性と流れ
2. 結論への論理的導線
3. データと主張の整合性
4. 読者視点での理解しやすさ"

### フェーズ7: 成果物の最終確認

**完成物の品質確認**

以下の成果物が完成しているか確認：
- [ ] 包括的リサーチレポート（30-40ページ）
- [ ] エグゼクティブサマリー（2-3ページ）
- [ ] 経営陣向けプレゼンテーション（10-15スライド）
- [ ] 実務担当者向け詳細資料（20-30スライド）
- [ ] インフォグラフィック集（5-10点）
- [ ] データ集とソース一覧

### 効率化のポイント

**並行処理の活用**

"複数の調査を並行実行してください：
- メイントピックの調査中に関連画像を生成
- データ収集と同時にビジュアル要素を準備
- レポート作成とプレゼンテーション構成を並行作業"

**テンプレートの活用**

"効率的な作業のためのテンプレートを作成してください：
- リサーチクエリのパターン集
- レポート構成のひな形
- プレゼンテーションのレイアウト基準"

**進捗管理**

"プロジェクト進捗を可視化してください：
- 各フェーズの完了状況
- 残り作業時間の見積もり
- ボトルネックの特定と対策"

### 応用可能な分野

このワークフローは以下の調査にも応用できます：

- **市場参入調査**: 新規事業の機会分析
- **競合分析**: 業界動向と競合戦略調査
- **技術トレンド**: 新技術の動向と影響分析
- **規制対応**: 法規制変更への対応策検討
- **投資判断**: M&A・投資機会の評価

### よくある課題と解決策

**情報過多による混乱**
→ 重要度によるフィルタリングと構造化整理

**調査の方向性迷い**
→ 初期の目的とスコープの明確化を徹底

**品質のばらつき**
→ チェックリストによる品質基準の統一

**時間超過**
→ 各フェーズの時間制限設定と進捗モニタリング
